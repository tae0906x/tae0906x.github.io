HTML describes the structure of a web page semantically and originally included cues for the appearance of the documaent.

HTML elements are the building blocks of HTML pages. with HTML constructs, images and other objects, such as interactice forms, may be embedded into the rendered page. It provides a means to create structured documents by denoting structural semantice for text such as heading, paragraphs, lists, links,quoter and other items.
